# SYNFIG_PLUGINS

This plugin requires GI (GObject Introspection). Please make sure it is installed.
install  pi:[ https://pypi.org/project/PyGObject/](https://packaging.python.org/en/latest/tutorials/installing-packages/)
or you can ask chatGpt

## what you get

     - add_display_param
     - split_bline
     - set_bone_with
     - center_bline
     - scale_bline_2x
     - clip_bline
     - mirror_bline
     - del_double_pointbline
      

<details><summary>Descriptions</summary>
  
### add_display_param

add param display in bone

### split_bline

set all param <split_radius> to active and <split_angle> to nonactive

### set_bone_with

set param <width> and <tipwidth>

### center_spline

to move offset your spline to center screen
set your layer group with name 'greyed'

### scale_spline_2x

to scale your spline 2x

### clip_bline

to <param name="blend_method"> to ONTO
your layers set descriptions to name 'clip'

### mirror_bline

to mirror spline at center off point
set your group of spline with name 'mirror'

## del_double_pointbline

will delete double point of bline


</details>
