# SYNFIG_PLUGINS

This plugin requires GI (GObject Introspection). Please make sure it is installed.
install  pi:[ https://pypi.org/project/PyGObject/](https://packaging.python.org/en/latest/tutorials/installing-packages/)
or you can ask chatGpt

## what you get

     - add_display_param
     - split_bline
     - set_bone_with
     - center_bline
     - scale_bline_2x
     - clip_bline
     - mirror_bline
     - del_double_pointbline
      

<details><summary>Descriptions</summary>
  
### add_display_param

add param display in bone

### split_bline

set all param <split_radius> to active and <split_angle> to nonactive
